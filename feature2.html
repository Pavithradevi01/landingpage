CSS

CSS (Cascading Style Sheets) is a language designed to simplify the process of making web pages presentable. It allows you to apply styles to HTML documents by prescribing colors, fonts, spacing, and positioning. The main advantages are the separation of content (in HTML) and styling (in CSS) and the same CSS rules can be used across all pages and not have to be rewritten. HTML uses tags, and CSS uses rule sets. CSS styles are applied to the HTML element using selectors. Cascading in CSS refers to how styles are applied to elements based on priority rules. When multiple CSS rules target the same element, the browser decides which style to apply by following the cascading order:inline styles, internal styles, and external stylesheets.

Some of the CSS properties are listed below:
  • color
  • letter-spacing
  • text-align
  • text-decoration
  • list-style
  • border-color
  • border-bottom
  • border-style
  • font-size
  • font-weight
The main use of CSS is:
  • Separation of Content and Style
  • Consistency
  • Faster Loading Times
  • Ease of Maintenance
  • Animations and Effects
  • Print-Friendly Pages
